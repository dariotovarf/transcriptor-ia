# 🎙️ Transcriptor IA

Aplicación web que permite:
- Subir audio o video
- Transcribir con IA (Whisper)
- Detectar idioma
- Traducir automáticamente al español

## 🛠️ Tecnologías
- FastAPI
- Whisper
- FFmpeg
- Python

## 🚀 Cómo ejecutar

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
